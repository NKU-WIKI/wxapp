## 🔧 技术实现

### CSS Filter 颜色转换
为了准确渲染`#6d28d9`紫色，使用了以下CSS filter：
```css
filter: brightness(0) saturate(100%) invert(27%) sepia(51%) saturate(2878%) hue-rotate(262deg) brightness(104%) contrast(97%)
```

这个filter能够将SVG图标精确转换为目标紫色。
